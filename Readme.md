# 🚀 CI/CD Pipeline using AWS CodeDeploy (No SSH Deployment)

## 📌 Overview

This project demonstrates a modern CI/CD pipeline using **Jenkins, AWS S3, AWS CodeDeploy, and EC2**.

It follows a fully automated, AWS-native deployment approach where **no SSH access is required for deployment**.

---

## 🧭 Architecture

```text
Developer → GitHub → Jenkins → S3 → AWS CodeDeploy → EC2
